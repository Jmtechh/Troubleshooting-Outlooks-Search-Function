<h1>Troubleshooting-Outlooks-Search-Function</h1>


<h2>Description</h2>
The aim of this lab is to demonstrate how to fix the search bar tool in Outlook if it is not working.

<h2>Utilities Used</h2>

- <b>Microsoft 365</b> 

<h2>Environments Used </h2>

- <b>Windows 10</b>
- <b>Outlook</b> 

<h2>Program walk-through:</h2>

First you click on file and then options. And then you go to the search function tab to click on indexing options.

<img src="https://i.imgur.com/0Djke3T.png" height="80%" width="80%" alt="Disk Sanitization Steps"/>

Next you click on the advanced button in indexing options.

<img src="https://i.imgur.com/V4NSIae.png" width="80%" alt="Disk Sanitization Steps"/>

Finally you click on the rebuild button. This should fix the issue.

<img src="https://i.imgur.com/9VVN5Hb.png" width="80%" alt="Disk Sanitization Steps"/>
